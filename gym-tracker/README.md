# Daywise Workout Routine

A gym routine planner that lets you build weekly workouts day by day.

## Features

- Plan workout exercises for each day of the week
- Set exercise name, sets, reps, weight, and notes
- Toggle completion for each exercise
- Switch between days using weekly tabs
- Clear all exercises for the selected day or the full week
- Saves routine data in browser `localStorage`

## Usage

1. Open `index.html` in a browser.
2. Choose a day and add routine exercises for that day.
3. Use weekly tabs to view or update other days.
4. Mark items complete as you finish them.

## PWA Install

- Run the app from a local server (for example `Live Server` in VS Code).
- Open the app in Chrome or Edge.
- Use the browser install prompt or tap the `Install App` button.
- The app can then be launched like a native app on mobile devices.
